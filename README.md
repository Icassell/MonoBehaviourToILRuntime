## V1.0

**简介**

ET_Mini是针对Unity3D的热更方案。
接入方便，追求的目标是一键化让新项目、老项目直接获得完善的热更能力！

**方案结构**

双热更，既包括在Android平台方便好用的Mono热更，还有更适合在IOS平台使用的ILRuntime热更。

自动化检测、分析工具，用于检测旧项目中，不适合热更的错误写法。

此外还包含很多可以提升代码运行效率的测试案例。

**开发环境**

Unity3D 2019.3.4f1

ILRuntime 1.64

XAsset 4.0

VS 2019

**使用步骤**

1、将你需要热更的脚本扔进本框架的Assets/Hotfix路径下

2、点击检测工具，排查需要调整的事项

3、进行资源打包（包含代码资源）

你的游戏就可以热更了，就是这么简洁。
